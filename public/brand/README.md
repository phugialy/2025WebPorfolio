# Phugialy Brand Package

## Recommended use
- Header/mobile nav: `phugialy-mark-light-on-dark.svg`
- Hero/footer: `phugialy-logo-full-light-on-dark.svg`
- Light backgrounds: use the `dark-on-light` versions
- Browser/app icons: favicon and icon PNG files
- LinkedIn/blog previews: `phugialy-social-card.png`

## Brand colors
- Yellow: #FFC400
- Black: #08090B
- White: #F7F7F5
- Gray: #A8ABB2

## Notes
- SVG is preferred for the website.
- PNG files have transparent backgrounds unless they are app/social assets.
- Keep clear space around the monogram.
- Do not stretch or recolor the logo.
